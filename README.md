# 📊 Grade Calculator (C++)

## 📌 Overview
This project is a console-based **Grade Calculator** developed in C++. It calculates the final grade of a student based on weighted components such as assignments, tests, exam, and participation.

The program demonstrates the use of **functions**, **modular programming**, and **basic input/output operations** in C++.

---

## 🚀 Features
- Accepts user input for different grade components  
- Applies predefined weightage to each component  
- Calculates final grade accurately  
- Displays detailed breakdown of weighted scores  
- Simple and easy-to-use console interface  

---

## 🛠️ Technologies Used
- **Language:** C++  
- **Concepts:** Functions, Modular Programming  
- **Input/Output:** Console-based  

---

## 📂 Project Structure
- `main.cpp` – Contains the complete program logic  

---

## ⚙️ How It Works
- User enters marks for:
  - Programming Assignments (15%)  
  - Chapter Tests (50%)  
  - Final Exam (30%)  
  - Class Participation (5%)  
- The program calculates weighted scores  
- Final grade is computed by summing all components
